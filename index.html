Introduction

This is a Z Shell script for using Z Shell as a CGI Script.

Usage

Use

source /path/to/zcu.zsh

at the begining of your script.

HTML Tags

General rules

We categoried HTML tags as 3 types:

  • Block. eg. <div>
  • OneLine. eg. <h1>
  • SelfColse. eg. <img>

Commands are related to a HTML tag, and is FirstLetterCapitalized to avoid commands with same names, such as <tr> and tr

Arguments can be given as

  • --{attr}={value} to make attributes of HTML tag.
  • A --{tag} changed tag name.( Not Recommended )
  • An argument not begin with -- is contents if needed.(Bug: if content begin with --)

Block

Use {Tag} and end{Tag}. For example

Div --class=test
endDiv

yields

<div class="test">
</div>

Oneline

For example

P --id=foo "Hello"

yields

<p id="foo">Hello</p>
